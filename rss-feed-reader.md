# RSS Feed Reader

A sleek and responsive **RSS Feed Reader** built with HTML, CSS, and JavaScript. Paste any RSS feed URL to instantly fetch and display the latest news in interactive cards.

## Features
- Load news from any RSS feed URL
- Responsive design with dark theme
- Interactive feed cards with hover effects
- Shows title, brief description, and link to full article
- Error handling for invalid URLs

## Usage
1. Open `index.html` in your browser
2. Paste a valid RSS feed URL
3. Click **Load Feed** to view articles

## Technologies
- HTML, CSS, JavaScript
- AllOrigins API for fetching RSS feeds without CORS issues
